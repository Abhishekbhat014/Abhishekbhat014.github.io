E-Commerce Portal

HTML5 CSS3 JavaScript

About The Project

This is a modern, fully responsive frontend portal designed for a large-scale e-commerce platform. The project emphasizes blazing-fast load times, semantic HTML structures for better SEO, and a highly interactive DOM manipulated with Vanilla JavaScript.

Key Features

  • Dynamic Rendering: Product catalogs are rendered dynamically from a JSON mock database using JavaScript, mimicking a real REST API integration.
  • Interactive Cart: Features a fully functional shopping cart that persists state using `localStorage`, allowing users to add, remove, and update quantities.
  • Responsive Design: Built using modern CSS Grid and Flexbox to ensure a seamless shopping experience from wide desktop monitors down to mobile viewports.
  • Performance First: Zero heavy JavaScript frameworks were used, leading to incredible Lighthouse scores for performance and accessibility.

Challenges & Solutions

Managing the state of the shopping cart across multiple browser tabs without a centralized state manager like Redux proved tricky. I solved this by implementing a custom event listener that triggers updates across tabs using the `storage` event in JavaScript, keeping the cart synchronized everywhere seamlessly.
