Private Vault

Flutter Google Drive API Hive OAuth

About The Project

Private Vault is a highly secure Flutter application designed for managing and storing personal photos and videos directly on Google Drive. It focuses heavily on user privacy and data security, integrating a seamless flat design aesthetic for a superior user experience.

Key Features

  • Google Drive Integration: Uses Google's OAuth2 API for authenticating users silently and uploading their personal media directly to their private cloud storage.
  • Local Caching: Implements Hive/SQLite to locally cache image metadata, providing lightning-fast load times when scrolling through the gallery without re-downloading files.
  • Encryption: Ensures that all local thumbnails and cached files are securely encrypted before being stored on the device.
  • Mandatory App Lock: Protects the app with a secure startup password barrier before allowing entry into the vault.

Challenges & Solutions

One of the major challenges was dealing with Google Sign-In silent authentication failures across app restarts. This was resolved by properly implementing the Singleton architecture for the AuthService and handling the edge cases of expired OAuth tokens.
